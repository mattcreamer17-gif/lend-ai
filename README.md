# Lendai working note

A single-page working note prepared for Lendai leadership (Yair Benyamini, Erez Dricker, Boaz Leviatan). The page carries the fit-analysis pitch in Lendai's own visual vocabulary (Inter, #B157FF purple, #011D3F navy) and routes to a 30-minute call with Matt Creamer, Eduba's CRO.

Intended URL path: `/for/lend-ai`

## Files

- `index.html` - structure and content
- `styles.css` - Lendai-branded styling (Inter, purple + navy, flat, no gradients beyond subtle accents)
- `logo.svg` - Lendai's official logo from the brand pack

## Notes

- No JavaScript. Single HTML file plus one stylesheet.
- Inter is loaded from Google Fonts because Lendai's own site serves it via Next.js scaffolding.
- Mobile-tested at 375px.
- Eduba appears only in the footer ("Prepared by Eduba. eduba.io") per the sprint rules.
- All external URLs are real anchors with `target="_blank"` and `rel="noopener"`.
- Calendly CTA points to `https://calendly.com/thecro-eduba/30min`.
